# real-time-server

### Description

A node app that does something.

### Installation

* Clone repo
* `yarn install`
* `yarn dev` to run dev env
* `yarn test` to run tests

### Dependencies

* Mocha and Expect.js for tests
* Puppeteer for UI testing
* Node-Fetch API client
* Express
* Dotenv
* Body parser
